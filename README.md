# Risk by Team7

Our implementation of the war/strategy game, Risk.

### Running

1. Download `Team7_Sprint_3`
2. Change to that directory
3. Run `$ java -Dprism.order=sw -jar ./Team7.jar`

> **Note**: MacOS users should leave out the JVM option

### Tools

- Java 8
- JavaFX.

### JVM Options

1. `-Dprism.order=sw`
